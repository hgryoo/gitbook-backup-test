담당 개발자가 Pull Request를 생성하고, Review 준비가 되었을 경우 “Submit a fix” 버튼을 눌러 “REVIEW IN PROGRESS” 상태로 변경합니다. 해당 Pull Request에 대하여 Review를 완료하고 반영\(merge\)되면 Github hook을 통해 JIRA에서 자동으로 “REVIEWED” 상태로 변경됩니다.
